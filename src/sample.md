# Papier, votre lecteur Markdown

Fini de lire du Markdown **balisé**. Ici tout est _mis en page_ proprement, et vous pouvez éditer directement dans le texte, comme dans Notion.

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Cmd + O` | Ouvrir un fichier |
| `Cmd + P` | Fichiers récents |
| `Cmd + F` | Rechercher dans le document |
| `Cmd + E` | Verrouiller / déverrouiller l'édition |
| `Cmd + S` | Enregistrer |
| `Cmd + \` | Afficher le sommaire |
| `Cmd + Maj + L` | Thème clair / sombre |

## Ce qui est bien rendu

- Les **titres** et la hiérarchie
- Les listes à puces et numérotées
- Les citations et le code
